# SportLoto
 SportLoto
Спортлото 6 из 49. Создайте программу, моделирующую получение 6 различных чисел из 49. Получение любого целого числа на интервале 1–49 равно возможно.
Ниже представлен код программы для 100 итераций, выполненный на языке программирования C#. Для того, чтобы установить 1000 или 10000 итераций, необходимо в цикле for поменять число 100 на 1000 или 10000.
